# 🌳 完整科技树设计方案

> **人类来时路 — The Path of Humanity**
> 以**核心材料与能源技术**定义时代，每项科技必须解锁具体内容（建筑/物品/配方）

---

## 设计原则

1. **以材料/能源定时代** — 每个时代的名字和特征由其标志性材料和能源技术定义
2. **去buff化** — 删除所有纯增益科技（如"效率+10%"），每项科技必须解锁至少一个具体内容：建筑`(B)`、物品`(I)`、配方`(R)`
3. **逻辑递进** — 前置→后置关系反映真实历史的技术依赖
4. **横向可选** — 同一时代内的分支科技可自由选择开发顺序

---
