# 🏆 人类来时路 — The Path of Humanity

**从原始采集到星辰大海** — 一款受《Factorio》《RimWorld》启发的文明演化模拟游戏。

> 带领一群原始人类，从零开始采集资源、建造家园、发展科技，在荒野中生存并走向繁荣。

**当前版本**: Alpha 0.1 | **引擎**: Godot 4.6

---

## 🎮 核心玩法

### 1️⃣ 生存与发展
- **管理定居者** — 每个定居者有独立的属性（力量、智力、敏捷等）、技能（采矿、建造、制作等）和需求（饱食度、精力、舒适度）
- **采集资源** — 派遣定居者采集木材、石头、浆果、矿石等自然资源
- **满足需求** — 保证定居者的饱食度和精力，否则他们会效率低下甚至生病

### 2️⃣ 建造系统

| 类别 | 建筑举例 |
|------|----------|
| 🪓 **采集** | 伐木屋、采石场、铁矿坑 |
| 🔧 **生产** | 工作台、熔炉、锯木厂、窑炉 |
| 🏠 **居住** | 帐篷、房屋 |
| 📦 **存储** | 储物架、仓库 |
| 🔬 **研究** | 研究台 |
| 🛡️ **防御** | 墙壁 |
| 🔥 **基础设施** | 篝火、道路 |

### 3️⃣ 科技树

通过研究台消耗资源解锁科技，逐级攀升：

```
原始工具 ─→ 建筑构造 ─→ 金属加工 ─→ 高级冶金
                  ├→ 烹饪技术
                  ├→ 木工技术
                  ├→ 石工技术
                  └→ 科学研究
```

### 4️⃣ 制作系统

| 建筑 | 可制作 |
|------|--------|
| **工作台** | 木板、石斧、石镐、火把 |
| **熔炉** | 铁锭、铜锭 |
| **灶台** | 熟肉、蔬菜汤 |
| **锯木厂** | 高效木板加工 |
| **窑炉** | 砖块 |

---

## 🎯 游戏目标

1. **生存** — 保证定居者有食物和住所
2. **扩张** — 建造更多建筑，扩大聚居地
3. **发展** — 研究科技，解锁更高级的建筑和配方
4. **繁荣** — 吸引更多定居者，最终走向工业时代

## 🏗️ 快速上手

1. 进入游戏后已有 **3 名定居者**
2. 观察地图上的 **树木**🌲、**浆果丛**🫐、**矿石**⛏️ 等资源
3. 点击 **「建造」** → 选择 **「工作台」** 放置
4. 建造 **「储物架」** 存储资源
5. 建造 **「帐篷」** 提供住所
6. 研究 **「原始工具」** 解锁石斧石镐
7. 逐步发展，扩张文明！

## 💻 开发 & 运行

### 环境要求
- [Godot 4.6+](https://godotengine.org/)
- VS Code + Godot Tools 扩展（可选，提供调试支持）

### 运行方式

```bash
# 方式一：Godot 编辑器打开
godot project.godot

# 方式二：VS Code 按 F5
# 确保已安装 Godot Tools 扩展，launch.json 已配置
```

### F5 调试（VS Code）

已内置调试配置：
- **▶ 运行游戏** — 直接启动
- **🐞 调试游戏** — 启动并附加调试器（支持 GDScript 断点）

## 🛠️ 技术特性

- **区块化世界生成** — 按需生成区块，支持无限地图
- **运行时纹理生成** — 无需外部图像文件，代码绘制所有纹理
- **昼夜系统** — 基于现实时间的昼夜循环
- **模块化系统架构** — 建筑、制作、科技系统独立解耦
- **数据驱动设计** — 所有物品/建筑/配方/科技集中定义
